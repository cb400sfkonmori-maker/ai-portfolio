# 🏎️ AI Cost Consultant (F1 Edition)
## 製造・飲食業向け 原価管理AIチャットボット

群馬県藤岡市を拠点とする「F1好きの鬼原価管理コンサルタント」を再現したAIデモアプリです。
Google Gemini APIを活用し、ビジネスのコスト管理を**F1（フォーミュラ1）のレース戦略**に例えてアドバイスします。

### 💡 特徴
* **Persona Design:** 単なる応答ボットではなく、「地元愛」と「F1マニア」という強烈な個性（システムプロンプト）を実装。
    * 例：「利益率はラップタイムと同じ」「無駄な経費は燃料の積みすぎ」などと独自の表現で回答します。
* **Serverless Architecture:** バックエンドサーバーを構築せず、HTML/JavaScriptのみで完結する軽量設計。
* **Security:** APIキーにHTTPリファラー制限をかけ、クライアントサイドでも安全に運用できる構成を実現。

### 🛠️ 使用技術
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **AI Model:** Google Gemini 1.5 Flash (via Gemini API)
* **Hosting:** GitHub Pages
